# HANSARD INTEL
## WA Parliament Opposition Research & Intelligence Platform

Production-grade parliamentary intelligence system for the WA Labor Party Premier's Office.

- **Backend**: FastAPI on Railway
- **Database**: PostgreSQL + pgvector on Railway  
- **Frontend**: React 18 + TypeScript, served from Railway
- **AI**: Anthropic Claude API
- **Scraper**: Daily at 6am AWST via APScheduler
